<strong>Lanzar una actualización:</strong> Para lanzar una actualización, se carga el archivo <em>upgrade.html</em> y el instalador de la actualización con el nombre <em>Upgrade.exe</em>. Esto abrirá una pestaña nueva en la app Attendant Zoom. Se debe cargar el archivo <em>upgrade.html</em> a http://www.goattendant.com/attendant-zoom/upgrade.html, esta es la página que se verá en Attendant Zoom.
<br><br>
<strong>¿Cómo se deciden las versiones de Attendant Zoom?</strong> La <em>e</em> al inicio de las versiones significa <em>edición</em>. Estas van de forma ascendente, después le sigue el año que para 2022 sería <em>22</em>. Después le sigue en la estación del año, por la cantidad de días que tiene la estación, para otoño sería <em>89.9</em>, si hay alguna actualización se le añade el código.<br><br>
